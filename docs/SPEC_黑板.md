# 黑板定义

## Fact（事实）

- **定义**：硬边界 / 已验证真值。只读。
- **追加规则**：仅当执行结果产出或用户以 `Fact+:` 显式确认时方可追加
- **标注**：每条 Fact 必须标注 (来源 / 置信度)
- **Human Gate 特权**：可通过 `Fact-` 降格或标记无效（非真正删除，而是状态标记）

## Intent（意图）

- **定义**：当前轮次自主声明的假设或方向
- **三类**：
  - 待验证：有明确预期产出，可被证伪
  - 待探索：方向性的、开放式的探查
  - 待决策：需要在候选项中择一，必须给出决策标准
- **用户否决权**：通过 `修正Intent:` 指令

## Hint（线索）

- **定义**：环境原始输入，包括用户给出的约束、偏好、禁忌、新数据、新上下文
- **性质**：Hint 是未经解读的素材，权重由角色自行判断
- **升格**：Hint 经验证后可被升格为 Fact
  - Auditor 检测黑板 Hint 是否被 Worker 引用
  - Auditor LLM 标记 suggest_promote_to_fact
  - Manager 评估：检查内容是否已在 Facts 中
  - 升格：Hint → Fact（source: "hint_promoted"）

## Round（轮次）

- **定义**：当前轮次计数器
- **规则**：每轮结束后 +1
- **用途**：多轮趋势分析（低谷检测、EI追踪）
